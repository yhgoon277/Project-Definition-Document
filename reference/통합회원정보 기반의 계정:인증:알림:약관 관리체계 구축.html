과제 개요

고객 여정채널 운영/경험
업무 영역회원 및 서비스 환경 설정 기능 설계
과제 목적

T4s에 분산된 계정·인증·알림·약관·설정 기능을 통합 제공하여,

고객이 가입·인증·정보관리·수신설정까지 계정 전반을 쉽고, 일관된 경험으로 셀프 관리할 수 있도록 하기 위함

문제 정의 요약현 상태
  • T4s(티월드/멤버십/티다/우주)별로 계정·인증·알림·약관·설정 기능이 분산 운영
  • 서비스별 상이한 인증 방식 및 정책으로 일관된 계정 경험 제공 미흡
  • 회원정보, 알림, 약관 동의 및 설정 기능이 각 채널별로 개별 관리
Pain Point
  • 계정/인증/약관/알림/설정 기능이 분산되어 일관된 관리 어려움
  • 서비스별 인증 방식 상이로 로그인/인증 과정의 번거로움 증가
  • 약관 동의 및 알림 수신 설정 상태를 통합적으로 확인/관리하기 어려움
  • 회원정보 변경 및 설정 경로가 분산되어 여러 화면 이동이 필요
  • 알림이 채널별로 분산되어 정보 누락 또는 과다 수신 발생
  • 로그인 이전에는 기능과 혜택 확인이 어려움
Root Cause
  • 채널별 계정 및 인증 체계의 분리 운영
  • 회원/약관/알림/설정 데이터의 통합 관리 구조 부재
  • 서비스 단위로 설계된 기능 구조로 고객 중심 통합 관리 미흡
  • 인증 및 설정 정책의 표준화 부족
미해결 시 Risk
  • 고객 계정 관리 피로도 증가 및 기능 탐색 시간 증가
  • 인증/설정 관련 문의 증가로 상담 의존도 확대
  • 개인화 서비스 확장 및 데이터 활용 한계
  • 채널 간 경험 불일치로 인한 고객 불만 및 이탈 가능성 증가
  • 운영/개발 중복 구조로 인한 비효율 지속


과제 정의

회원 가입·로그인·탈퇴·인증, 회원정보 관리, 약관 동의, 알림, 설정을 담당한다.

채널 통합의 기반이 되는 계정·인증 체계를 포괄한다.

주요 기능

1CI 기반 통합 계정 관리

단일 CI 기반으로 가입/로그인/탈퇴/계정 변경까지 계정 통합 관리

가입 → 계정 생성 → 계정 연동 → 계정 유지

2통합 로그인 및 세션 관리

가입/로그인 → 인증 수단 선택(SSO/PASS/소셜) → 인증 수행 → 세션 유지 및 계정 진입

  • 다양한 로그인 수단 제공: SSO, PASS 인앱, 소셜 로그인
  • 세션 유지: 재로그인 없이 서비스 연속 이용
  • 비로그인 상태 → 로그인 전환 유도 (혜택/정보 기반)
3인증 모듈 공통화 및 간소화

서비스 진입/주요 액션 → 상황별 인증 필요 판단 → 인증 수행 → 서비스 이용

  • 인증 모듈 공통화: 생체/PASS/소셜 인증
  • 상황 기반 인증: 가입/변경/결제 등 단계별 차등 적용
  • 인증 최소화: 반복 인증 방지 및 간소화
4회원정보 통합 관리 (CRUD)

회원정보 진입 → 정보 조회 → 수정/추가 → 저장 및 반영

  • 회원정보 CRUD: 프로필/배송지/연락처/결제수단
  • 통합 관리: 채널별 분산 정보 일원화
  • 변경 이력 반영 및 최신 상태 유지
5

통합 알림 센터 및 수신 제어

알림 수신 → 알림 센터 진입 → 유형별 확인 → 수신 설정 변경 → 후속 행동 연결

  • 통합 알림 센터: 채널별 알림 통합 제공
  • 수신 설정: 채널/유형별 ON/OFF 및 권한 설정
  • 행동 연계: 알림 → 관련 화면 이동 및 액션 유도
6

약관 통합 관리 및 이력 조회

서비스 이용 → 약관 확인 → 동의/거부 → 이력 조회/철회/재동의

  • 약관 통합 관리: 서비스별 약관 일원화
  • 동의 이력 관리: 동의 상태 및 변경 이력 조회
  • 약관 구조 단순화: 이해하기 쉬운 구성 제공
7

통합 설정 및 계정 제어 기능

설정 진입 → 항목 선택 → 설정 변경 → 즉시 반영

  • 통합 설정: 알림/개인정보/접근성/언어 등
  • 개인화 설정: 기본값 및 선호 설정 관리
  • 권한 제어: 서비스별 접근 및 이용 권한 관리
8

비로그인 상태 가치 제공 및 전환 유도

비로그인 상태 → 혜택/정보 탐색 → 로그인 필요 시점 인지 → 로그인/가입 전환

  • 비로그인 가치 제공: 멤버십/혜택 미리보기
  • 전환 유도: 로그인 시 추가 혜택/기능 안내
  • 자연스러운 진입: 강제 로그인 없이 탐색 가능
