# 과제정의서 발췌 — #8 고객 주문경험 혁신 프로세스 재설계

(Claude Design 프로젝트에 첨부하는 상세 자료)

- 문서번호: TK-CH-X001
- 고객여정: 채널 운영/경험
- 업무영역: 주문
- 카테고리: 주문/구매 (order)

## 목적

1. 표준화된 공통 주문 프로세스 수립으로 채널·상품 무관 일관된 경험 제공
2. 모듈형·재사용 가능한 설계로 확장성 확보

## 현재 상태 (AS-IS)

- 상품/유형별 다른 주문 단계와 인증 방식
- 앱/웹 이원화로 복잡성 증가
- 비대면 완결률 저조

## Pain Points

- 복잡한 단계와 반복 입력
- 채널별 다른 흐름으로 혼란
- 비대면 완결률 저조로 오프라인 의존

## 근본 원인

- 표준화된 공통 주문 프로세스 부재
- 앱/웹 분리 운영
- 고객정보 재활용 체계 미흡

## 핵심 기능 (features)

1. 로그인 상태 정의 및 비로그인 탐색
2. 상품 선택/구성 모듈 ★ 파일럿 채택
3. 주문 정보 입력 표준화 ★ 파일럿 채택
4. 약관 동의 및 인증
5. 결제 연동 ★ 파일럿 채택
6. 상태 관리 및 개통 ★ 파일럿 채택
7. 온보딩 및 자동연결 ★ 파일럿 채택

(★ 4개를 Feature 1~4로 묶어 10개 화면으로 구성함)

## 지향점 및 기대효과

| 지향점 | 설명 | 기대효과 |
|---|---|---|
| 표준 모듈 구조 설계 | 가입·인증·약관·결제·개통 공통 모듈화, 재조합 가능 | 프로세스 중복·운영 복잡도 감소, 신규 상품 빠른 적용 |
| AI 기반 주문 지원 기반 | 주문 단계·상태·예외를 표준 스키마로 정의 | AI 주문 도우미 확장 가능, 일관된 주문 경험 |
| 입력량 최소화 | "길이"보다 "생각할 양"과 "입력할 양" 축소 | 주문 완료율 향상, 피로·망설임 감소 |
| 고객 정보 재활용 | 보유 정보·직전 입력값·인증 결과 자동 반영, 이탈 후 복원 | 재입력 피로 없이 빠른 주문 |
| 오류 발생 자체를 줄이는 구조 | 반복 이슈를 사전 검증·자동화로 재설계 | 셀프개통 완료율·주문 완결률 향상 |
| 막힘 순간 즉시 이해 | 실패 원인·영향·다음 행동을 진행 중 바로 제공 | 실패→이탈 비율 감소 |
| AI 주문 도우미 | 단계별 시각 안내, 개통 가이드, 상담 연결까지 AI 지원 | 몰라서 불안한 상황 방지 |
| 셀프개통 핵심 경험 | 셀프개통을 독립 여정으로 설계(단계형 UX, 오류 복구, 도움 연결) | 주문 완료↔실사용 시작 단절 해소 |
| 주문의 끝 = 실사용 시작 | 신청 완료 후 결과 이해·개통·혜택·서비스 연결까지 하나의 흐름 | 가입 후 즉시 사용 가능, 비용 불신 해소 |

## 주요 프로세스 및 기능 (전 여정 11단계)

### 1. 로그인 상태 정의
비로그인/로그인 상태 식별 · 최근 주문 이어하기 · 비로그인 허용 범위 정의

### 2. 비로그인 탐색
상품 목록/상세 조회 · 요금제/단말/혜택 비교 · 예상 납부 금액 시뮬레이션 · AI 주문 도우미 기반 상품 설명

### 3. 상품 선택/구성 모듈 ★
메인상품 선택(단말/요금제/가입유형) · 추가상품·사은품 선택 · 비교/추천/시뮬레이션 · 개인화 시뮬레이션

### 4. 주문 시도 및 로그인 전환
주문 버튼 클릭 시 로그인 요청 · 간편 로그인/본인 확인 · 이전 선택값 복원

### 5. 자격/조건 확인
회선/명의/연령/약정 기반 자격 판별 · 불가 사유 및 대안 경로 안내

### 6. 주문정보 입력 ★
배송지/연락처/청구지 입력 · 로그인 정보 기반 자동 입력 · 포인트/쿠폰/할인 등록 · 이탈 후 이어하기

### 7. 약관동의·인증
약관 통합 동의 · 본인/명의/법정대리인 인증 · 인증 결과 재사용 · 간편 인증 수단 확대

### 8. 결제 ★
결제수단 선택 · 실납부액 최종 확인 · 신청 완료 및 주문번호 발급

### 9. 상태관리 ★
주문/배송/개통 상태 조회 · 보류 사유 안내 · 재시도/취소/담당 부서 연결

### 10. 개통 ★
유심/eSIM/기기변경 개통 · 단계형 개통 가이드 · 오류 복구/재시도 · AI 개통 도우미

### 11. 온보딩 ★
개통 완료 확인 · 티월드/멤버십/구독 자동 연동 · 혜택/할인 변화 안내 · 다음 해야 할 일 제시
